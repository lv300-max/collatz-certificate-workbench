Overview For Humans

What is being checked?

The packet checks whether a finite set of symbolic residue lanes has descent certificates. A lane certificate says that every number in that lane eventually drops below itself, after which strong induction can take over.

What does B mean?

For a symbolic formula T^m(n) = (3^o n + b) / 2^c, a positive gap 2^c - 3^o gives a threshold B. Above that threshold, the lane descends. Below it, the direct bridge checks the small cases.

Why the direct bridge matters

Every final threshold in the packet lands at or below 200001. The direct bridge verifies every odd number from 3 through 200001 with zero failures.

What should reviewers attack?

The arithmetic reports passed internally. The higher-value review is structural: residue exhaustiveness, quotient abstraction validity, cap-stopped row closure, local-key state completeness, and no sampled evidence in the proof path.

Math Map For Reviewers

Symbolic lane formula

The raw Collatz iterate is tracked as T^m(n) = (3^o n + b) / 2^c. The proof-critical comparison is exact: gap = 2^c - 3^o. If gap > 0, then the lane descends for every n > B, where B = (b + gap - 1) // gap. 

T^m(n) < n
(3^o n + b) / 2^c < n
b < n(2^c - 3^o)
n > ceil(b / (2^c - 3^o))

Exact arithmetic

Integer powers, integer gap comparison, and integer ceiling are used for proof checks. Floating delta fields are diagnostic only.
